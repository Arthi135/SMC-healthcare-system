mart Medicine System
📌 Project Overview

The Smart Medicine System is a web-based application designed to provide information about hospital services and healthcare facilities. The system helps users easily access details about doctors, services, hospital staff, ambulance services, bed availability, and patient care.

This project aims to create a simple and user-friendly platform where patients can view hospital information and services online.

🎯 Objectives

To provide hospital information in an organized digital format.

To help patients easily access doctor and service details.

To improve communication between hospitals and patients.

To provide quick access to services such as ambulance and bed availability.
